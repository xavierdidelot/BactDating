Example of using CreDating on the data of Harris et al (2010)

Xavier Didelot

2017-10-12

Initialisation

library(CreDating)
library(ape)
set.seed(0)
data(harris2010)
plot(harris2010$tree, show.tip.label = F)
axisPhylo(backward = F)

Root-to-tip analysis

Lets do a root-to-tip regression analysis:

res=roottotip(harris2010$tree,harris2010$dates)

Analysis

We run CreDating as follows:

res=credating(harris2010$tree,harris2010$dates,nbIts = 50000,model=3,updateRate=1)
plot(res,show.tip.label = F)

Plot with credibility bars:

plot(res,'treeCI',show.tip.label = F)

We can see what the MCMC traces look like:

  plot(res,'trace')
